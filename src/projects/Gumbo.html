Gumbo

The Problem:

At events, everyone captures memories on their own phones—but those photos and videos rarely get shared or organized.
📸 Great moments are scattered across devices.
📤 Sharing them later is messy, inconvenient, or forgotten.
😩 Hosts and guests miss out on reliving the full experience together.

The Solution – Gumbo:

🎉 Gumbo is a collaborative event memory app that brings all your guests’ photos and videos into one shared memory wall.

📱 Hosts create an event and share a code. Guests use the code to upload, view, and download media in real-time.

💾 With editing tools and download options, it’s the easiest way to turn moments into memories—together.

Key Features

  • 🥳 Event-Based Media Walls

    Create or join events using a unique invite code and instantly access a shared memory wall where all guests can upload photos and videos in real-time.

  • 🎨 In-App Media Editing Tools

    Apply filters, crop images, and trim videos directly within the app—no need for external editors. Make every memory look just right before sharing.

  • 🔐 Private & Controlled Access

    Only invited guests with the event code can view or contribute to an event’s memory wall—keeping event content secure and private.

  • ⬇️ Media Download & Sharing Options

    Host-controlled download permissions allow guests to save their favorite moments and easily share them across other platforms.

Application Overview

Create an Event

Follow our easy step-by-step process to create your event. Start by clicking ‘Create an Event’, fill out all the necessary details, and generate a unique access code. Share this code with your guests for an exclusive invite to your memory wall.

Upload Content

Capture the essence of your event by sharing photos and videos. Simply click ‘Upload’, select your media, and let the sharing begin! Watch as your memory wall fills with the vibrant colors and emotions of your event.

View Shared Content

Once your event is over, the fun doesn’t have to end! Revisit your memory wall anytime to relive those precious moments. Engage with the content, leave comments, and keep the memories alive.

Architecture

Technologies Used

1. Flutter

We use Flutter to build a beautiful, consistent, and high-performance mobile app for both Android and iOS. By using a single codebase, we ensure faster development cycles, lower maintenance overhead, and a consistent user experience across platforms.

2. AWS API Gateway

API Gateway acts as the entry point for all client-server interactions. It securely routes requests from the app to backend services, ensuring smooth communication between the front end and the serverless backend.

3. AWS Lambda

Our backend logic is powered by AWS Lambda, a serverless computing service. This eliminates the need for managing servers and allows Gumbo to scale on demand based on user activity. This is majorly used for generating Payment URL for Stripe.

4. Stripe

Stripe is integrated to handle donations. It enables secure, flexible, and globally trusted payment processing—supporting both one-time and recurring donations via credit/debit cards.

5. Webhooks

We use webhooks for real-time event communication—such as receiving instant payment status updates from Stripe or notifying the system of a completed transaction. This keeps the app responsive and up-to-date.

6. Admin Portal

The Gumbo admin portal is a web-based tool used by internal teams to manage content, review analytics, provide transaction details, and maintain platform integrity. It provides complete backend visibility and control over the platform’s operations.

7. Firebase

Firebase is integrated into our application to manage user authentication securely and efficiently. It also powers real-time notifications, allowing users to receive instant updates and alerts, thereby enhancing the overall responsiveness and interactivity of the app.

Architecture

Technologies Used

1. Flutter

We use Flutter to build a beautiful, consistent, and high-performance mobile app for both Android and iOS. By using a single codebase, we ensure faster development cycles, lower maintenance overhead, and a consistent user experience across platforms.

2. AWS API Gateway

API Gateway acts as the entry point for all client-server interactions. It securely routes requests from the app to backend services, ensuring smooth communication between the front end and the serverless backend.

3. AWS Lambda

Our backend logic is powered by AWS Lambda, a serverless computing service. This eliminates the need for managing servers and allows Calorii to scale on demand based on user activity. This is majorly used for generating Payment URL for Razorpay.

4. Razorpay

Razorpay is integrated to handle donations. It enables secure, flexible, and globally trusted payment processing‚Äîsupporting both one-time and recurring donations via credit/debit cards.

5. Webhooks

We use webhooks for real-time event communication‚Äîsuch as receiving instant payment status updates from Razorpay or notifying the system of a completed transaction. This keeps the app responsive and up-to-date.

6. Admin Portal

The Calorii admin portal is a web-based tool used by internal teams to manage blogs, YouTube videos, manage payments, review analytics and maintain platform integrity. It provides complete backend visibility and control over the platform‚Äôs operations.

7. Firebase

Firebase is integrated into our application to manage user authentication securely and efficiently. It also powers real-time notifications, allowing users to receive instant updates and alerts, thereby enhancing the overall responsiveness and interactivity of the app.

8. Calendly

Calendly is integrated into our application to streamline scheduling and appointment management. It allows users to book meetings effortlessly by syncing with their calendars, reducing back-and-forth communication and improving overall efficiency in coordinating availability.

9. YouTube

The YouTube API is leveraged to embed and manage video content within our application. It enables dynamic access to YouTube‚Äôs vast video library, supports playback customization, and allows seamless integration of multimedia elements to enrich user engagement.

Willow

The Problem:

👩‍👧‍👦 As a busy parent, finding reliable services for your children—like babysitting, camps, lessons, or health services—can be stressful and time-consuming.

💬 You don’t always know who to trust, how to book, or how to keep track of everything in one place.

The Solution – Willow:

📲 Willow is your one-stop app to find, book, and pay for trusted child-related services.

🔍 Browse top-rated providers, ⭐ read real parent reviews, and 📅 schedule with ease.

✅ Both parent and provider confirm service start and completion, ensuring transparency and reliability.

💳 Pay seamlessly via Stripe, Apple Pay, or card—no cash, no stress.

Simple. Safe. Seamless.

Key Features

  • 🗂️ Multi-Category Booking System

    Allow users to easily browse and book from a wide range of services like babysitting, camps, health, and lessons—streamlined in one app.

  • ✅ Real-Time Service Confirmation Flow

    Dual confirmation buttons ("Start Service" / "Complete Service") for both parents and providers ensure secure, transparent, and verified service delivery.

  • 💳 Integrated Payments & Scheduling

    Smooth, secure payments via Stripe, Apple Pay, and more—combined with real-time scheduling for a hassle-free booking experience.

  • ⭐ Verified Reviews & Ratings System

    Build trust with parent-generated reviews and ratings, helping users choose reliable, quality service providers.

Architecture

Technologies Used

1. Flutter

We use Flutter to build a beautiful, consistent, and high-performance mobile app for both Android and iOS. By using a single codebase, we ensure faster development cycles, lower maintenance overhead, and a consistent user experience across platforms.

2. AWS API Gateway

API Gateway acts as the entry point for all client-server interactions. It securely routes requests from the app to backend services, ensuring smooth communication between the front end and the serverless backend.

3. AWS Lambda

Our backend logic is powered by AWS Lambda, a serverless computing service. This eliminates the need for managing servers and allows Willow to scale on demand based on user activity. This is majorly used for generating Payment URL for Stripe.

4. Stripe

Stripe is integrated to handle donations. It enables secure, flexible, and globally trusted payment processing—supporting both one-time and recurring donations via credit/debit cards.

5. Webhooks

We use webhooks for real-time event communication—such as receiving instant payment status updates from Stripe or notifying the system of a completed transaction. This keeps the app responsive and up-to-date.

6. Admin Portal

The Willow admin portal is a web-based tool used by internal teams to manage content, review analytics, provide transaction details, and maintain platform integrity. It provides complete backend visibility and control over the platform’s operations.

7. Firebase

Firebase is integrated into our application to manage user authentication securely and efficiently. It also powers real-time notifications, allowing users to receive instant updates and alerts, thereby enhancing the overall responsiveness and interactivity of the app.
